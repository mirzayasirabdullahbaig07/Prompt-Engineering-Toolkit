# 📌 About

The **Missing Context Detector** is a browser-based prompt engineering tool that analyzes AI prompts to identify missing information that could reduce the quality of AI-generated responses.

The application evaluates prompts for essential context such as action verbs, audience, output format, measurable constraints, purpose, time references, and ambiguous language. It provides an overall context score along with detailed recommendations to help users create more complete and effective prompts.

Built entirely using **HTML**, **CSS**, and **Vanilla JavaScript**, the tool runs completely in the browser without requiring any frameworks, APIs, or backend services.

# 🎯 Context Validation

The detector checks whether your prompt contains:

- ✅ Clear Action
- ✅ Intended Audience
- ✅ Output Format
- ✅ Quantity or Constraints
- ✅ Context or Purpose
- ✅ Time Reference
- ✅ Unambiguous Language

It highlights any missing elements and suggests improvements for creating stronger AI prompts.
