# 🧠 Chain of Thought Builder

A modern and interactive web application that helps users generate **step-by-step reasoning prompts** using the **Chain of Thought (CoT)** prompting technique. This tool breaks complex problems into logical reasoning steps, making AI responses more structured, transparent, and easier to understand.

## 📖 Overview

Chain of Thought (CoT) prompting encourages AI models to solve problems by reasoning through intermediate steps before arriving at a final answer.

This application allows users to:

- Enter a system prompt
- Provide a user problem or question
- Generate structured reasoning steps
- View the final answer
- Maintain a history of generated prompts

The project is built entirely with **HTML, CSS, and Vanilla JavaScript** and features a clean, responsive user interface.

# ⚙️ How It Works

1. Enter a **System Prompt** (optional).
2. Enter a problem or question in the **User Problem** field.
3. Click **Generate Steps**.
4. View the generated reasoning process.
5. Review the final answer.
6. Access previous prompts from the history section.
