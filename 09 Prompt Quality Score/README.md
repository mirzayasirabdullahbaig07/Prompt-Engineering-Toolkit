# 📌 About

The **Prompt Quality Score** is an interactive browser-based prompt engineering tool designed to evaluate how effective an AI prompt is before using it with Large Language Models (LLMs).

The tool analyzes multiple quality dimensions—including **Clarity, Structure, Specificity, and Actionability**—and generates an overall quality score along with AI-powered insights to help users improve their prompts.

Built entirely with **HTML**, **CSS**, and **Vanilla JavaScript**, the application runs completely in the browser without requiring any APIs, frameworks, or backend services.

# 📊 Quality Metrics

The analyzer evaluates prompts using four core quality pillars:

- **Clarity** – Measures how clearly the prompt communicates its objective.
- **Structure** – Evaluates the organization and formatting of the prompt.
- **Specificity** – Checks whether the prompt includes sufficient details, numbers, or constraints.
- **Actionability** – Determines whether the prompt begins with clear action-oriented instructions.

These metrics are combined to generate an overall **Prompt Quality Score**.

# 🎯 Quality Levels

| Score | Quality |
|--------|----------|
| **81 – 100** | 🔥 Excellent Prompt Quality |
| **61 – 80** | 👍 Good Prompt |
| **41 – 60** | ⚠ Needs Improvement |
| **0 – 40** | ❌ Poor Prompt Quality |
