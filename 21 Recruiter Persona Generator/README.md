# 👥 Recruiter Persona Generator

An interactive AI-powered web application that generates structured **Recruiter Personas** based on industry, hiring role, company type, and recruitment goals. This tool helps HR professionals, recruiters, AI prompt engineers, and hiring managers create realistic recruiter identities for recruitment planning, AI prompting, talent acquisition, and hiring simulations.

## 💡 How It Works

1. Enter the industry.
2. Specify the hiring role.
3. Select the company type.
4. Define the hiring goal.
5. Click **Generate Recruiter Persona**.
6. Instantly receive:
   - Recruiter Profile
   - Hiring Strategy
   - Candidate Focus
   - Recruitment Dashboard
   - Decision Pipeline
   - Hiring Progress Visualization
