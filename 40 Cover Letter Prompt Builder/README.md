# 📄 Cover Letter Prompt Generator

A simple, responsive **Cover Letter Prompt Generator** built with **HTML, CSS, and Vanilla JavaScript**. This application helps users quickly create a structured and professional cover letter by entering basic information such as their name, desired job title, and skills.

Designed and developed by **Hamna Munir** and **Mirza Yasir Abdullah Baig**.

---

# ✨ Features

[coverletterpromptgenerator](https://coverletterpromptgenerator.netlify.app/)

---


# ✨ Features

* 📄 Generate professional cover letters instantly
* 👤 Personalize letters with applicant information
* 💼 Specify the job title or position
* 🛠️ Include custom skills and qualifications
* 📋 Automatically creates structured sections:

  * Applicant Information
  * Introduction
  * Skills & Experience
  * Why Me
  * Closing
* 💾 Stores the last 10 generated cover letters
* 📖 Reload previous letters from history
* 🎨 Clean and responsive user interface
* ⚡ Fast generation with no external APIs required
* 🧹 One-click clear functionality

---

# 📸 User Interface

The application is organized into five main sections.

## Header

Displays:

* Project title
* Developer information

---

## Input Section

Provide the following details:

* **Your Name**
* **Job Title**
* **Skills** (comma-separated)

Example:

```text
Name: Hamna Munir
Job Title: Software Engineer
Skills: HTML, CSS, JavaScript, Communication
```

---

## Action Buttons

Two primary actions are available:

* **Generate Cover Letter** – Creates a personalized cover letter.
* **Clear** – Resets all input fields and the generated output.

---

## Output Section

Displays a formatted cover letter including:

* Cover Letter Title
* Applicant Information
* Introduction
* Skills & Experience
* Why Me
* Closing Statement
* Applicant Signature

Each section is presented in an easy-to-read format for quick review and editing.

---

## History

The application automatically stores the most recent **10 generated cover letters**.

Users can click any history item to instantly restore and view a previously generated cover letter.

---

# 📋 Generated Cover Letter Structure

Every generated cover letter follows this format:

1. Applicant Information
2. Introduction
3. Skills & Experience
4. Why Me
5. Closing
6. Signature

This structure provides a strong starting point for job applications and can be customized further if needed.

---

# 🚀 How It Works

1. Enter your name.
2. Enter the job title you are applying for.
3. List your skills (comma-separated).
4. Click **Generate Cover Letter**.
5. Review the generated letter in the output section.
6. Access previous letters through the History panel.
