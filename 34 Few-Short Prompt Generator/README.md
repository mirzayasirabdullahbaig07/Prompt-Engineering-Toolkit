# 🚀 Few-Shot Prompt Generator

A modern and interactive web application that helps users create **Few-Shot AI Prompts** using example-based learning. The application enables users to define a task, provide sample input-output examples, and generate a structured prompt that guides AI models to produce more accurate and context-aware responses.

## 📖 Overview

Few-shot prompting is one of the most effective Prompt Engineering techniques where AI is provided with a few examples before processing a new input.

This application simplifies the prompt creation process by organizing:

- Task Description
- Example 1
- Example 2
- New Input
- Final Prompt Generation

into a clean and user-friendly interface.

# ⚙️ How It Works

1. Enter the AI task description.
2. Provide Example 1 (Input → Output).
3. Provide Example 2 (Input → Output).
4. Enter the new input that should be processed.
5. Click **Generate Prompt**.
6. Copy the generated prompt for use with AI tools.
