# 📌 About

The **Prompt Complexity Analyzer** is an interactive browser-based tool that evaluates how complex an AI prompt is by analyzing its length, instruction density, reasoning demand, constraints, and keyword distribution.

It provides an overall complexity score, detailed prompt statistics, visual progress indicators, keyword extraction, and intelligent optimization suggestions to help users create better prompts for Large Language Models (LLMs) such as ChatGPT, Claude, Gemini, and Microsoft Copilot.

Built entirely using **HTML**, **CSS**, and **Vanilla JavaScript**, the application runs completely in the browser without requiring any frameworks, APIs, or backend services.

# 🎯 Complexity Levels

The analyzer classifies prompts into four categories:

| Complexity Score | Level |
|------------------|-------|
| **0 – 39** | Basic |
| **40 – 59** | Moderate |
| **60 – 79** | Advanced |
| **80 – 100** | Expert |

These levels help users understand how detailed and sophisticated their prompts are before submitting them to an AI model.
