# 📌 About

The **AI Role Generator** is a lightweight browser-based tool that automatically generates multiple AI personas based on a user's task description. Assigning a specific role to an AI model helps produce more accurate, structured, and context-aware responses.

The tool instantly creates specialized AI roles such as strategists, prompt engineers, context analyzers, quality controllers, and optimizers to improve prompt engineering workflows.

Built entirely using **HTML**, **CSS**, and **Vanilla JavaScript**, it runs completely in the browser without requiring any frameworks, APIs, or backend services.
