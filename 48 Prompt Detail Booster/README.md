# ✨ Overview

Prompt Detail Booster is a modern web application that converts short, vague prompts into highly detailed and professional AI prompts.

The application leverages Anthropic Claude to automatically enrich prompts by adding:

- Expert roles
- Context
- Objectives
- Output formatting
- Audience specifications
- Constraints
- Examples
- Quality criteria
- Usage recommendations

It also evaluates the enhanced prompt using multiple quality metrics and provides a before/after comparison.

---

# ✨ Features

## 📝 Prompt Enhancement

- Expand short prompts into professional prompts
- Multiple enhancement levels
- Domain-specific optimization

---

## 🎯 Three Detail Levels

### ⚡ Medium

- Role assignment
- Clear task
- Structured output

### 🔥 High

- Rich context
- Audience targeting
- Examples
- Step-by-step instructions

### 🚀 Ultra

Includes everything in High plus:

- Constraints
- Edge cases
- Quality criteria
- Advanced prompt engineering
- Expert-level instructions

---

## 🌍 Domain Selection

Choose from multiple domains including:

- General
- Technology
- Business
- Creative
- Education
- Science

---

## 📊 Prompt Quality Analysis

Each generated prompt includes:

- Overall Enhancement Score
- Detail Score
- Clarity Score
- Context Score
- Actionability Score

---

## 🔍 Before vs After Comparison

Compare:

- Original prompt
- AI-enhanced prompt

side by side.

## 💡 Prompt Suggestions

Automatically explains:

- What was added
- Why it improves the prompt
- Best practices for using the generated prompt

## 📜 History

- Save recently boosted prompts
