# 🧩 Prompt Formatter Tool

A modern, interactive Prompt Formatter built with **HTML, CSS, and JavaScript** that transforms raw prompts into professionally structured prompts for AI models like ChatGPT, Claude, Gemini, and GPT-4.

The application analyzes user input, assigns an AI role, improves prompt quality, provides formatting suggestions, and generates production-ready prompts in multiple styles.

---

## ✨ Features

- 🎯 Convert raw prompts into structured AI prompts
- 🤖 Supports multiple AI platforms
  - ChatGPT
  - Claude
  - Gemini
  - GPT-4
- 📝 Multiple formatting styles
  - Basic Format
  - Professional Format
  - Expert Format
  - API Style (JSON)
- 📊 Prompt quality score
- 📈 Before vs After comparison
- 📚 Prompt history
- 📋 One-click copy
- ⚡ Responsive modern UI
- 🎨 Beautiful purple-themed interface
- 🔄 Local formatter fallback when API is unavailable

# ⚙️ How It Works

1. Enter your raw prompt.
2. Select your target AI platform.
3. Choose a formatting style.
4. Click **Format Prompt**.
5. Review:
   - Prompt Quality Score
   - Improvements
   - Structured Prompt
6. Copy the generated prompt.

# 📖 Formatting Styles

### 📝 Basic Format

Simple role + task + instructions.

---

### 💼 Professional Format

Includes:

- Role
- Context
- Instructions
- Output Format

### 🧠 Expert Format
