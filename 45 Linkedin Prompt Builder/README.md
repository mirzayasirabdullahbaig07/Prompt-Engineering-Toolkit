# 📖 Overview

LinkedIn Prompt Generator is a modern web application that helps professionals instantly generate polished LinkedIn content with the power of AI.

Simply enter your achievement, choose a content type and tone, and the application generates professional content suitable for LinkedIn.

The project features an elegant responsive interface, real-time preview, prompt visualization, content history, and one-click copy functionality.

# ⚙️ How It Works

1. Enter a topic or achievement.
2. Select the content type.
3. Choose your preferred tone.
4. Add optional details.
5. Click **Generate Content**.
6. AI creates professional LinkedIn content.
7. Copy and publish directly on LinkedIn.
