# 📖 Overview

The **YouTube Prompt Generator** is a modern web application built for content creators who want to generate high-quality YouTube content using AI.

Simply enter your video topic and a few details such as niche, audience, tone, and format. The application generates:

- 🎯 SEO Optimized Titles
- 🎬 Hook
- 📝 Complete Script Outline
- 📄 Video Description
- 🖼 Thumbnail Idea
- 🏷 SEO Tags

The interface is designed with a clean two-panel workspace for maximum productivity and ease of use.

# ⚙️ How It Works

1. Enter your video topic.
2. Choose niche and audience.
3. Select format and tone.
4. Add optional key points.
5. Click **Generate Content**.
6. AI generates:

- Titles
- Hook
- Script
- Description
- Thumbnail
- Tags

7. Copy or save your generated content.
