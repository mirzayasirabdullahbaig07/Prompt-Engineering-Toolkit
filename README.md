## 📌 About The Project

The **Prompt Engineering Toolkit** is a comprehensive suite of **50 standalone web tools** designed to help developers, students, researchers, and AI enthusiasts craft, analyze, refine, and optimize prompts for large language models (LLMs).

Every tool is:
- **100% standalone** — a single `.html` file, no build step required
- **Fully functional** — real working logic, not just UI mockups
- **Professionally designed** — consistent white, lavender & purple color theme
- **Zero-dependency** — open in any browser, offline friendly

---

## ✨ Features

- 🔍 **Prompt Analysis & Scoring** — Evaluate quality, clarity, readability, and structure
- 🎭 **Persona Builders** — Generate expert, teacher, recruiter, developer & more AI personas
- ✍️ **Prompt Refiners** — Expand, simplify, clean, refine, and reformat prompts
- 🧪 **Testing & Comparison** — Sandbox, version tracker, and prompt comparator
- 🏗️ **Prompt Builders** — Purpose-built builders for blogs, emails, resumes, LinkedIn, YouTube, and more
- 🎓 **Advanced Techniques** — Zero-shot, few-shot, chain-of-thought prompt generators

## 🛠️ Tech Stack

```
Frontend Only — No frameworks, no build tools, no dependencies

HTML5          →  Structure & markup
Inline CSS3    →  Styling & animations (all self-contained)
Vanilla JS     →  Logic, interactivity & analysis algorithms
```

All tools run **entirely in the browser** — no server, no API, no internet required after loading.

## 🚀 Getting Started

### Option 1 — Open Directly (Recommended)
```bash
# Clone the repository
git clone https://github.com/Hamna-Munir/Prompt-Engineering-Toolkit.git

# Navigate to folder
cd Prompt-Engineering-Toolkit

# Open any tool in your browser
open "01. Prompt Analysis Tools.html"
```

### Option 2 — GitHub Pages
Each tool can be accessed directly via GitHub Pages at:
```
https://hamna-munir.github.io/Prompt-Engineering-Toolkit/01.%20Prompt%20Analysis%20Tools.html
```

### Option 3 — Download ZIP
Click **Code → Download ZIP** from the repository, unzip, and open any `.html` file.

## 🎨 Design Theme

All 50 tools follow a consistent, professional design language:

| Element | Value |
