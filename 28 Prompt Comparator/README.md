# 📊 Prompt Comparator

A modern and responsive web application that compares two AI prompts by analyzing their structure, word usage, and similarity score. The application provides detailed statistics, identifies unique and common words, and helps users evaluate and improve prompt quality for better AI interactions.

## 📌 Features

* 📄 Compare two AI prompts side by side
* 📊 Display word, character, and sentence counts
* 🎯 Calculate prompt similarity percentage
* 📈 Visual similarity progress bar
* 🟣 Identify unique words in each prompt
* 🔍 Display common words shared between prompts
* 🧹 One-click clear/reset functionality
* 📱 Fully responsive design
* 🎨 Modern purple-themed user interface
* ⚡ Built with pure HTML, CSS, and JavaScript

## ⚙️ How to Run

1. Clone the repository

```bash
git clone https://github.com/your-username/prompt-comparator.git
```

2. Open the project folder.

3. Launch `index.html` in your preferred web browser.

No installation or additional dependencies are required.

---

## 📖 How It Works

1. Enter the first prompt in **Prompt A**.
2. Enter the second prompt in **Prompt B**.
3. Click **Compare Prompts**.
4. Review:

   * Word statistics
   * Character count
   * Sentence count
   * Similarity score
   * Unique words
   * Common words
5. Use the **Clear** button to reset the comparison.
