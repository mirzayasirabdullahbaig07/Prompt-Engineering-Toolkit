# ✨ Prompt Cleaner Pro

Prompt Cleaner Pro is a modern web-based AI prompt optimization tool that analyzes, cleans, and improves user prompts. It removes filler words, identifies structural issues, calculates a prompt quality score, and generates a professional version of the prompt with detailed feedback.

## ⚙️ How It Works

1. Enter or paste a prompt.
2. Click **Clean & Analyze**.
3. The application:
   - Detects filler words
   - Finds prompt issues
   - Calculates a quality score
   - Cleans the prompt
   - Generates AI-powered suggestions
4. Copy the optimized prompt with one click.
