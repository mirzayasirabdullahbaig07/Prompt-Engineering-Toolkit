## 📖 Overview

The **Prompt Refiner** is a lightweight web application designed to improve user-written prompts before they are used with AI systems. It replaces vague wording with more precise language, improves clarity, and applies prompt engineering best practices to generate higher-quality AI responses.

---

## ✨ Features

* 📝 Enter any prompt for refinement
* ✨ Generates a clearer and more professional prompt
* 📊 Displays the original and refined prompts side by side
* 🧠 Shows the enhancements applied during refinement
* 🎨 Modern responsive UI with a clean purple-themed design
* ⚡ Instant processing using JavaScript
* 📱 Mobile-friendly layout

## 💡 How It Works

1. Enter a prompt into the input area.
2. Click **Refine Prompt**.
3. View:

   * 📌 Original Prompt
   * ✨ Refined Prompt
   * 📊 List of Improvements Applied
