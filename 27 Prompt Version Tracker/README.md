# 📖 About The Project

**Prompt Version Tracker** is a lightweight web application designed for AI engineers, prompt engineers, researchers, and content creators who frequently refine prompts for Large Language Models (LLMs).

Instead of losing previous prompt versions, this application allows users to save, organize, compare, and manage prompt iterations efficiently.

Every version is stored locally in the browser, enabling users to revisit previous prompts, compare improvements, and export their prompt history for backup or collaboration.

Whether you're working with **ChatGPT**, **Claude**, **Gemini**, **DeepSeek**, **Llama**, or any other LLM, Prompt Version Tracker helps streamline your prompt engineering workflow.

- 📂 Export prompt history as JSON

---

# 💡 How It Works

1. Enter your AI prompt.
2. Click **Save Version**.
3. The prompt is stored locally.
4. Browse your version history.
5. View details of any saved version.
6. Compare two versions side-by-side.
7. Export your complete prompt history as a JSON file.

# 📊 Dashboard

The application automatically tracks:

- 📚 Total Prompt Versions
- 🔢 Latest Version Number
- 📝 Total Words Across All Prompts

# 🔍 Version Comparison

Users can compare any two saved prompt versions to view:

- Total word count
- Difference in prompt length
- Preview of each prompt
- Easy side-by-side analysis

This helps identify how prompts evolve over time and supports iterative prompt engineering.
