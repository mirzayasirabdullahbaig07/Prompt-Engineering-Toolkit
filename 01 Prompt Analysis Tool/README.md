# 📌 About

The **Prompt Analysis Tool** is a browser-based application that helps users evaluate the quality of AI prompts before using them with Large Language Models (LLMs) such as ChatGPT, Claude, Gemini, and Copilot.

It analyzes prompts across multiple dimensions including clarity, specificity, readability, sentiment, structure, complexity, keywords, and overall quality score.

The tool is built entirely using **HTML**, **CSS**, and **Vanilla JavaScript**, requiring no external libraries or backend services.
