# 🚀 Zero-Shot Prompt Generator

A modern and interactive web application that helps users create effective **Zero-Shot AI prompts** without providing examples. The **Zero-Shot Prompt Generator** guides users through essential prompt components such as task description, target audience, tone, and output format to produce structured, AI-ready prompts.

---

## 📌 Features

- ✨ Generate Zero-Shot AI prompts instantly
- 🎯 Define clear task descriptions
- 👥 Specify target audience
- 🎨 Choose response tone
- 📄 Select output format
- 📊 Live prompt statistics
- 📋 Copy generated prompts
- 🧹 Clear all inputs
- 📱 Fully responsive design
- 🎨 Modern Purple Gradient User Interface

## 🚀 How to Use

1. Open the application in your web browser.
2. Enter the task description.
3. Specify the target audience (optional).
4. Select the desired tone.
5. Choose the preferred output format.
6. Click **Generate Prompt**.
7. Copy the generated prompt and use it with your favorite AI model.

---

## 💡 Example

### Input

**Task Description**

```
Explain Artificial Intelligence and its real-world applications.
```

**Target Audience**

```
University Students
```

**Tone**

```
Professional
```

**Output Format**

```
Article
```

---

### Generated Prompt

```
TASK:
Explain Artificial Intelligence and its real-world applications.

TARGET AUDIENCE:
University Students

TONE:
Professional

OUTPUT FORMAT:
Article

INSTRUCTIONS:
Provide a complete and accurate response.
Do not request examples.
Use only the information provided.
Maintain consistency throughout the response.
Generate the final output directly.
```
