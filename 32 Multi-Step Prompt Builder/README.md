# 🚀 Multi-Step Prompt Builder

A modern and interactive web application that helps users build powerful AI prompts using a guided multi-step approach. The **Multi-Step Prompt Builder** organizes prompt creation into structured sections such as role, task, context, constraints, output format, and examples, making prompt engineering easier and more effective.

---

## 📌 Features

- ✨ Build AI prompts step-by-step
- 👤 Define AI Role / Persona
- 🎯 Specify Goals and Tasks
- 📝 Add Context and Constraints
- 📂 Choose Output Format
- 💡 Include Examples for Better Responses
- 📊 Live Prompt Statistics
- 📋 Copy Generated Prompt
- 🧹 Clear All Inputs
- 📱 Fully Responsive Design
- 🎨 Modern Purple Gradient User Interface

## 🚀 How to Use

1. Open the application in your web browser.
2. Enter the AI goal or task.
3. Define the AI role or persona.
4. Provide relevant context.
5. Add any constraints or requirements.
6. Select the desired output format.
7. Include examples (optional).
8. Click **Generate Prompt**.
9. Copy the generated prompt for use with your preferred AI model.

---

## 💡 Example

### Input

**Goal**

```
Write a professional blog about Artificial Intelligence.
```

**Role**

```
Act as an experienced AI Content Writer.
```

**Context**

```
The article is intended for beginners who want to understand AI.
```

**Constraints**

```
Maximum 800 words.
Use simple language.
Include practical examples.
```

**Output Format**

```
Blog Post
```

---

### Generated Prompt

```
ROLE:
Act as an experienced AI Content Writer.

TASK:

(Optional user examples)
```
