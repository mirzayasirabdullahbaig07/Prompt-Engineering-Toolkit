# 👥 Customer Persona Prompt Builder

Create structured customer personas for AI-powered marketing prompts with ease. **Customer Persona Prompt Builder** helps marketers, entrepreneurs, prompt engineers, and content creators generate detailed customer persona prompts by defining product details, audience demographics, goals, and experience levels.

---

## 📌 Overview

Customer Persona Prompt Builder is a lightweight web application that generates comprehensive customer persona prompts for AI models like ChatGPT, Claude, Gemini, and other LLMs. It helps users build marketing-focused personas that include behavioral traits, motivations, pain points, and decision triggers for better prompt engineering and campaign planning.

---

## ✨ Features

- 👥 Generate AI-ready customer persona prompts
- 🎯 Define target audience by age and experience level
- 🛍️ Product or service-based persona creation
- 📈 Marketing goal integration
- 🧠 Automatic persona traits generation
- 💡 Includes motivations, behaviors, and pain points
- 🎨 Modern responsive interface
- ⚡ Lightweight and fast
- 📱 Mobile-friendly design
- 💻 Built using HTML, CSS & JavaScript

# 💡 How It Works

1. Enter the product or service.
2. Specify the target age group.
3. Define your marketing goal.
4. Select the customer experience level.
5. Click **Generate Persona Prompt**.
6. Copy the generated customer persona prompt for use with your favorite AI assistant.
