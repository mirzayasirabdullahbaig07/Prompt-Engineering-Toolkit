# 📧 Email Prompt Generator

A modern, responsive **Email Prompt Generator** that helps users create professional email prompts with an intuitive interface. The application collects email details, generates a structured AI prompt, previews the email, maintains generation history, and provides copy-to-clipboard functionality.

# ⚙️ How It Works

1. Enter the recipient name.
2. Enter the sender name.
3. Add the email subject.
4. Specify the recipient role (optional).
5. Add the organization (optional).
6. Describe the purpose of the email.
7. Include additional details (optional).
8. Select the desired email tone.
9. Click **Generate Email**.
10. Preview, copy, or revisit previous emails from history.
