# 📌 About

The **Prompt Readability Score** is a browser-based prompt engineering tool that evaluates how easy an AI prompt is to read and understand.

The tool analyzes readability using metrics such as sentence length, average word length, total word count, and an overall readability score. It also provides helpful suggestions for improving prompt clarity, making prompts more effective for Large Language Models (LLMs) like ChatGPT, Claude, Gemini, and Microsoft Copilot.

Built entirely with **HTML**, **CSS**, and **Vanilla JavaScript**, the application runs fully in the browser without requiring any APIs, frameworks, or backend services.

# 📊 Readability Metrics

The analyzer evaluates prompts using multiple readability indicators:

- Overall Readability Score
- Sentence Length
- Word Count
- Average Word Length
- Complexity Level
- Prompt Improvement Suggestions

These metrics help users create prompts that are easier for both humans and AI systems to interpret.
