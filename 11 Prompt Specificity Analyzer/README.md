# 📌 About

The **Prompt Specificity Analyzer** is an interactive browser-based prompt engineering tool that evaluates how specific and detailed an AI prompt is before submitting it to a Large Language Model (LLM).

The analyzer measures prompt quality using multiple specificity indicators such as **Detail Level**, **Precision**, **Numeric Depth**, and an overall **Specificity Score**. It also identifies vague words, highlights prompt tokens, and provides AI-generated suggestions for making prompts clearer and more precise.

Built entirely using **HTML**, **CSS**, and **Vanilla JavaScript**, the application runs completely in the browser without requiring APIs, frameworks, or backend services.

# 📊 Analysis Metrics

The analyzer evaluates prompts using four major dimensions:

- **Detail Level** – Measures the amount of descriptive information provided.
- **Precision** – Detects vague or ambiguous wording.
- **Numeric Depth** – Identifies the use of numbers, limits, and measurable constraints.
- **Overall Specificity Score** – Combines all metrics into a comprehensive evaluation.

---

# 🎯 Specificity Levels

| Score | Level |
|--------|------------------------|
| **81 – 100** | 🎯 Highly Specific |
| **61 – 80** | ✅ Good Specificity |
| **41 – 60** | ⚠ Moderate Specificity |
| **0 – 40** | ❌ Low Specificity |
