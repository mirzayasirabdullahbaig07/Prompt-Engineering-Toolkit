# 💻 Developer Persona Generator

An interactive AI-powered web application that generates structured **Developer Personas** based on a developer's role, experience level, technology stack, and focus area. This tool helps developers, AI prompt engineers, educators, recruiters, and technical teams quickly create realistic engineering identities for AI prompting, software development scenarios, and technical documentation.

## 💡 How It Works

1. Enter the developer role.
2. Select the experience level.
3. Specify the technology stack.
4. Define the primary focus area.
5. Click **Generate Developer Persona**.
6. Instantly receive:
   - Developer Persona Name
   - Role Type
   - Experience Level
   - Technology Stack
   - Focus Area
   - Engineering Profile Summary
