## 📖 Overview

**Prompt Focus Enhancer** is a modern web application designed to improve poorly written or vague prompts into clear, structured, and highly focused prompts for AI assistants.

The application analyzes the user's input, identifies weaknesses, assigns a focus score, and generates a professional prompt using multiple enhancement strategies. When available, it uses **Claude AI** for advanced prompt engineering, while also providing a powerful local enhancement engine as a fallback.

# 🎯 Focus Modes

## 🔍 General Focus

Perfect for everyday prompts.

## ⚡ Strict Focus

Designed for highly accurate responses.

- Enforces concise writing

# 📋 How It Works

1. Enter your prompt.
2. Select a focus mode.
3. Click **Enhance Prompt**.
4. The system analyzes your prompt.
5. Weaknesses are detected.
6. Focus score is calculated.
7. A professional prompt is generated.
8. Copy the enhanced prompt with one click.
