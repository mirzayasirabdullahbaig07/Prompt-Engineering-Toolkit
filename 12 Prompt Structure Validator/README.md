# 🏗️ Prompt Structure Validator

Validate the structural quality of AI prompts by analyzing their organization, logical flow, action clarity, and overall prompt construction. This lightweight web application helps prompt engineers create prompts that are easier for AI models to understand and execute.

---

## 📌 Overview

Prompt Structure Validator evaluates whether a prompt follows a strong structural pattern. It checks if the prompt includes:

- Clear action verbs
- Logical flow
- Proper prompt length
- Basic prompt organization

The tool provides a visual structure flow with validation indicators to help users improve prompt quality.

---

## ✨ Features

- ✅ Detects action verbs in prompts
- ✅ Evaluates logical connections
- ✅ Checks prompt structure and organization
- ✅ Visual workflow validation
- ✅ Action, Logic Flow & Order scores

## 💡 How It Works

1. Enter your AI prompt.
2. Click **Validate**.
3. The tool evaluates:
   - Action clarity
   - Logical flow
   - Prompt organization
4. Review the validation scores and structure flow.
5. Improve your prompt based on the feedback.
