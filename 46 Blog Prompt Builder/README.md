# 📝 Blog Prompt Generator

A modern, responsive **AI-powered Blog Prompt Generator** that helps users create complete blog strategies with a clean and intuitive interface. Simply enter a blog topic, choose your preferences, and generate SEO-friendly blog titles, outlines, introductions, keywords, and more.

## ⚙️ How It Works

1. Enter your blog topic.
2. Add optional details such as:
   - Author Name
   - Blog Type
   - Writing Tone
   - Target Audience
   - Key Points
3. Click **Generate Blog**.
4. The application creates:
   - Blog Titles
   - Meta Description
   - Introduction
   - Blog Outline
   - Conclusion
   - SEO Keywords
   - Writing Tip
