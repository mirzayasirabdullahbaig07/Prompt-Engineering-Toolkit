## 📌 Overview

The **Interviewer Persona Builder** helps users quickly create professional interviewer personas based on hiring requirements. It generates structured interviewer profiles that can be used for AI prompting, HR automation, interview preparation, recruitment workflows, and prompt engineering.

Simply provide:

- Job Role
- Experience Level
- Interview Style
- Company Type

The application instantly generates a detailed interviewer persona with hiring strategies and evaluation focus areas.

## 📸 Application Workflow

1. Enter the job role.
2. Select the experience level.
3. Specify the interview style.
4. Enter the company type.
5. Click **Generate Interviewer Persona**.
6. Review the generated interviewer profile, strategy, skills, and evaluation focus.
