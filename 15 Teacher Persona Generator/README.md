## 📖 Overview

The **Teacher Persona Generator** helps educators, instructional designers, prompt engineers, and AI enthusiasts create structured teacher personas for AI-powered educational applications.

Users simply provide:

- 📚 Subject
- 🎓 Education Level
- 👨‍🏫 Teaching Style
- 🎯 Teaching Goal

The application instantly generates a professional AI teacher profile with expertise, teaching capabilities, and instructional methodology.

## 🎯 How It Works

1. Enter the **Subject**.
2. Select the **Education Level**.
3. Provide the **Teaching Style**.
4. Define the **Teaching Goal**.
5. Click **Generate Teacher Persona**.
6. View the generated AI teaching profile, expertise summary, and capabilities.
