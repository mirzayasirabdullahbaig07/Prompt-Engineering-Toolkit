# 📌 About

The **Prompt Ambiguity Detector** is a browser-based prompt engineering tool designed to identify vague words, unclear references, and ambiguous language that can reduce the quality of AI-generated responses.

The tool analyzes prompts for commonly used ambiguous terms, calculates an overall clarity score, highlights problematic words, and provides useful statistics to help users write more precise and effective prompts for Large Language Models (LLMs).

Built entirely using **HTML**, **CSS**, and **Vanilla JavaScript**, the application runs completely in the browser without requiring external libraries, frameworks, APIs, or backend services.

# 🎯 What the Tool Detects

The Prompt Ambiguity Detector identifies commonly used vague expressions such as:

- Pronouns without clear references
- Generic words
- Indefinite quantities
- Unclear expressions
- Weak descriptive language
- Non-specific instructions

Examples include:

- it
- this
- that
- they
- something
- anything
- everything
- stuff
- things
- maybe
- perhaps
- some
- many
- various

Detecting these words helps create prompts that are more specific, detailed, and easier for AI models to understand.
