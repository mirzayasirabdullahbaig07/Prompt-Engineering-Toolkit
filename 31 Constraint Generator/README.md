# 🚀 Constraint Generator

A modern and interactive web application that helps users generate useful constraints and prompt rules for AI tasks. The Constraint Generator simplifies prompt engineering by automatically suggesting practical constraints that improve AI-generated responses.

---

## 📌 Features

- ✨ Generate AI prompt constraints instantly
- 📝 Create structured prompt rules
- 📊 Live statistics dashboard
- 📋 Copy generated constraints
- 🧹 Clear all inputs with one click
- 📱 Fully responsive design
- 🎨 Modern Purple Gradient UI
- ⚡ Built using pure HTML, CSS & JavaScript

---

## 🚀 How to Use

1. Open the application in your browser.
2. Enter a task or prompt.
3. Click **Generate Constraints**.
4. View suggested constraints and prompt rules.
5. Copy the generated rules or clear the workspace.

---

## 💡 Example

### Input

```
Write a blog post about Artificial Intelligence.
```

### Generated Constraints

- Maximum 500 words
- Use professional language
- Include practical examples
- Maintain logical structure
- Add a concise conclusion
- Avoid repetitive content
- Target beginner-level audience
- Use proper grammar
- Focus on the topic
- Organize content with headings
