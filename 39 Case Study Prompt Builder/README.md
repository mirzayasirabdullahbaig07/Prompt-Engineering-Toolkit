# 📊 Case Study Prompt Generator

A simple, responsive **Case Study Prompt Generator** built using **HTML, CSS, and Vanilla JavaScript**. This application helps users quickly generate a structured case study outline based on a given topic, making it useful for students, educators, researchers, and professionals.

Designed and developed by **Hamna Munir** and **Mirza Yasir Abdullah Baig**.

---

# ✨ Features

* 📊 Generate structured case studies from any topic
* 📝 Automatically creates:

  * Introduction
  * Problem Statement
  * Analysis
  * Solution
  * Outcome
  * Conclusion
* 💾 Stores the last 10 generated case studies
* 📖 Restore previous case studies from history
* 🎨 Clean and responsive user interface
* ⚡ Instant generation without external APIs
* 🧹 One-click clear functionality

---

# 📸 User Interface

The application consists of four main sections.

## Header

Displays:

* Project title
* Developer information

---

## Input Section

Enter the topic of your case study.

Example:

```text
Impact of Social Media on Students
```

---

## Action Buttons

Two primary actions are available:

* **Generate Case Study** – Creates a structured case study based on the provided topic.
* **Clear** – Clears the input field and generated output.

---

## Output Section

Displays a formatted case study containing:

* Case Study Title
* Introduction
* Problem Statement
* Analysis
* Solution
* Outcome
* Conclusion

Each section is presented in an easy-to-read card layout.

---

## History

The application automatically saves the most recent **10 generated case studies**.

Users can click any history item to instantly reload that case study into the output panel.

---

# 📋 Generated Structure

Every generated case study follows this format:

1. Introduction
2. Problem Statement
3. Analysis
4. Solution
5. Outcome
6. Conclusion

This consistent structure makes it suitable for academic assignments, presentations, and project planning.

---

# 🚀 How It Works

1. Enter a case study topic.
2. Click **Generate Case Study**.
3. The application automatically builds a structured case study template.
4. View the generated content in the output section.
5. Previous generations are saved in the History panel for quick access.
