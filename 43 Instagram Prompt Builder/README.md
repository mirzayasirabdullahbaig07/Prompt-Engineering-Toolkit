# 📸 Instagram Prompt Generator

An AI-powered **Instagram Prompt Generator** that helps users create engaging Instagram content in seconds. The application generates captions, reel scripts, carousel ideas, and story content using AI while providing a beautiful live preview, prompt history, and one-click copy functionality.

## 📖 Overview

The Instagram Prompt Generator is designed for content creators, influencers, businesses, and marketers who want to generate high-quality Instagram content quickly.

Users simply enter their content details, choose the desired content type and tone, and the application creates AI-powered Instagram content with hashtags and a call-to-action.

# ⚙️ How It Works

1. Enter your Instagram topic.
2. Select your niche.
3. Choose your target audience.
4. Select the language.
5. Choose the content type.
6. Select the content tone.
7. Add optional instructions.
8. Click **Generate Content**.
9. View the AI-generated Instagram content.
10. Copy the generated content instantly.
