# 📋 Evaluation Criteria Builder

A modern and responsive web application for automatically generating evaluation criteria and grading rubrics for academic projects, assignments, software applications, and AI-based solutions. The tool enables educators, trainers, and evaluators to quickly build structured assessment criteria, assign weightages, and customize evaluation standards.

# ✨ Features

* 📋 Automatically generate evaluation criteria
* 🏆 Built-in grading rubric (Excellent, Good, Average, Below Average)
* ➕ Add custom evaluation criteria
* ❌ Remove criteria instantly
* 📊 Automatic total criteria count
* 📈 Dynamic total weight calculation
* 🎯 100-mark evaluation framework
* 📱 Fully responsive design
* 🎨 Modern purple-themed UI
* ⚡ Fast and lightweight (Pure HTML, CSS & JavaScript)


## Run the Project

1. Open the project folder.

---

# 📖 How It Works

1. Enter the project or assignment description.
2. Click **Generate Criteria**.
3. Review the automatically generated evaluation criteria and weight distribution.
4. Add custom criteria with your preferred weight percentage.
5. Remove criteria whenever needed.
6. Monitor the total number of criteria and overall weight in real time.

# 🏆 Built-in Grading Rubric

* **Excellent (90–100):** Outstanding implementation and quality.
* **Good (75–89):** Meets requirements with minor improvements needed.
* **Average (60–74):** Acceptable implementation with noticeable improvements required.
* **Below Average (<60):** Significant improvements required.
